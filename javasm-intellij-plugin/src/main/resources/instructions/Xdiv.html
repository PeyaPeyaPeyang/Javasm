Division Instructions

{link:idiv} - 0x6c (108)
{link:ldiv} - 0x6d (109)
{link:fdiv} - 0x6e (110)
{link:ddiv} - 0x6f (111)

Purpose

Divides the second-top operand on the stack by the top operand and pushes the result. For integer types, division by zero throws ArithmeticException. For floating types, division by zero results in Infinity or NaN per IEEE 754.

Operands: None
Format: {instruction}
Examples:
  • idiv
  • ddiv
Stack Effect:
  • Before: ..., value1, value2 (value1 is dividend, value2 is divisor)
  • After: ..., result
