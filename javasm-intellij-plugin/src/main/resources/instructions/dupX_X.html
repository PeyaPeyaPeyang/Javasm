Stack Duplication Instructions

{link:dup} - 0x59 (89)
{link:dup_x1} - 0x5a (90)
{link:dup_x2} - 0x5b (91)
{link:dup2} - 0x5c (92)
{link:dup2_x1} - 0x5d (93)
{link:dup2_x2} - 0x5e (94)

Purpose

Duplicates one or two values on the operand stack and optionally inserts them deeper into the stack.

Operands: None
Format: {instruction}
スタック効果:
  • {link:dup}
    Before: ..., value1（value1 is category 1）
    After: ..., value1, value1
  • {link:dup_x1}
    Before: ..., value2, value1（both are category 1）
    After: ..., value1, value2, value1
  • {link:dup_x2}
    Form 1（すべて category 1）:
    Before: ..., value3, value2, value1
    After: ..., value1, value3, value2, value1
    Form 2:
    Before: ..., value2, value1（value2 is category 1, value1 is category 2）
    After: ..., value1, value2, value1
  • {link:dup2}
    Form 1:
    Before: ..., value2, value1（both are category 1）
    After: ..., value2, value1, value2, value1
    Form 2:
    Before: ..., value1（category 2）
    After: ..., value1, value1
  • {link:dup2_x1}
    Form 1（すべて category 1）:
    Before: ..., value3, value2, value1
    After: ..., value2, value1, value3, value2, value1
    Form 2:
    Before: ..., value2, value1（value2 is category 1, value1 is category 2）
    After: ..., value1, value2, value1
  • {link:dup2_x2}
    Form 1（すべて category 1）:
    Before: ..., value4, value3, value2, value1
    After: ..., value2, value1, value4, value3, value2, value1
    Form 2:
    Before: ..., value3, value2, value1（value3 is category 2, value2 and value1 are category 1）
    After: ..., value1, value3, value2, value1
    Form 3:
    Before: ..., value3, value2, value1（value3 is category 2, value2 and value1 are category 1）
    After: ..., value2, value1, value3, value2, value1
    Form 4:
    Before: ..., value2, value1（both are category 2）
    After: ..., value1, value2, value1
Examples:
  • dup
  • dup_x1
  • dup2 (with long or two ints)
Verification:
  • Verification requires that the operand stack can accept the required values and resulting duplicates at the point of this instruction.
Runtime Exceptions and Errors:
  • VerifyError: if the operand stack shape or category does not match any valid form for the instruction.
