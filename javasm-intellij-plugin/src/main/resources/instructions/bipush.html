Push Byte Value

{link:bipush} - 0x10 (16)

Purpose

Pushes a signed byte value (−128 to 127) onto the operand stack as an int. This instruction is used for values outside the range handled by iconst_m1 to iconst_5.

The immediate byte operand is sign-extended to a 32-bit int before being pushed.

Operands:
  • byte: 8-bit signed immediate value to push onto the stack.
Format: bipush <byte>
Examples:
  • bipush 100
  • bipush -128
Stack Effect:
  • Before: ...
  • After: ..., value (category-1 int)
Verification:
  • Verification requires that the operand stack can accept an int value at the point of this instruction.
Notes:

For values between −1 and 5 inclusive, use the corresponding iconst_m1 to iconst_5 instructions instead of bipush.

Runtime Exceptions and Errors:

This instruction does not throw any instruction-specific runtime exceptions.
