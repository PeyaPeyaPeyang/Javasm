Push Byte Value

{link:bipush} - 0x10 (16)

Purpose

Pushes a signed byte value (−128 to 127) onto the operand stack as an int. This instruction is used for values outside the range handled by iconst_m1 to iconst_5.

Operands:
  • byte: 8-bit signed immediate value to push onto the stack.
Format: bipush <byte>
Examples:
  • bipush 100
  • bipush -128
Note:

For values between −1 and 5 inclusive, use the corresponding iconst_m1 to iconst_5 instructions instead of bipush.

Stack Effect:
  • Before: ...
  • After: ..., value (int)
