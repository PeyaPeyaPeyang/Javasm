Load Constant Instructions

{link:ldc} - 0x12 (18)
{link:ldc_w} - 0x13 (19)
{link:ldc2_w} - 0x14 (20)

Purpose

Loads a constant from the constant pool onto the operand stack. The constant can be a primitive value, a string, or a class reference.

Operands:
  • ConstantIndex: An index into the constant pool table. The type of the constant determines which instruction to use:
    • ldc: For int, float, or String constants.
    • ldc_w: Same as ldc, but supports a wider (16-bit) index.
    • ldc2_w: For long or double constants.
Format:
  • ldc <ConstantIndex>
  • ldc_w <ConstantIndex>
  • ldc2_w <ConstantIndex>
Examples:
  • ldc 10 (loads constant at index 10, e.g., a string or integer)
  • ldc_w 300 (loads constant at index 300, supports larger indices)
  • ldc2_w 5 (loads a long or double constant at index 5)
Stack Effect:
  • ldc, ldc_w
    Before: ...
    After: ..., value (the constant value is pushed onto the stack)
  • ldc2_w
    Before: ...
    After: ..., value2, value1 (for long or double, which occupy two stack words)
