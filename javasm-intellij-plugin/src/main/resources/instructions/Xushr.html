Unsigned Shift Right Instructions

{link:iushr} - 0x7c (124)
{link:lushr} - 0x7d (125)

Purpose

Shifts a value to the right logically (unsigned) by a specified number of bits.
The shift amount is masked: only the lowest 5 bits are used for {link:iushr}, and the lowest 6 bits for {link:lushr}.

  • {link:iushr}: logically unsigned right-shifts an int by an int amount
  • {link:lushr}: logically unsigned right-shifts a long by an int amount
Operands: None
Format: {instruction}
Examples:
  • iushr
  • lushr
Stack Effect:
  • iushr
    Before: ..., value (int), shift (int)
    After: ..., result (int)
  • lushr
    Before: ..., value (long), shift (int)
    After: ..., result (long)
