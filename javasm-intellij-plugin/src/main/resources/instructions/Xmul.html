Multiplication Instructions

{link:imul} - 0x68 (104)
{link:lmul} - 0x69 (105)
{link:fmul} - 0x6a (106)
{link:dmul} - 0x6b (107)

Purpose

Multiplies the top two operands on the operand stack and pushes the result. The operation respects the type of the instruction.

Operands: None
Format: {instruction}
Examples:
  • imul
  • dmul
Stack Effect:
  • Before: ..., value1, value2
  • After: ..., result
