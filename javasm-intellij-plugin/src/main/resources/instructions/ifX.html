Conditional Branch Instructions

{link:ifeq} - 0x99 (153)
{link:ifne} - 0x9a (154)
{link:iflt} - 0x9b (155)
{link:ifge} - 0x9c (156)
{link:ifgt} - 0x9d (157)
{link:ifle} - 0x9e (158)

Purpose

Compares the top value on the operand stack with zero and branches to the specified label based on the result.

  • {link:ifeq}: Branches if the value is equal to zero.
  • {link:ifne}: Branches if the value is not equal to zero.
  • {link:iflt}: Branches if the value is less than zero.
  • {link:ifge}: Branches if the value is greater than or equal to zero.
  • {link:ifgt}: Branches if the value is greater than zero.
  • {link:ifle}: Branches if the value is less than or equal to zero.
Operands:
  • label: The target label to branch to if the condition is met.
Format:
  • ifeq <label>
  • ifne <label>
  • iflt <label>
  • ifge <label>
  • ifgt <label>
  • ifle <label>
Examples:
  • ifeq zero_label
  • ifne non_zero_label
  • iflt negative_label
  • ifge non_negative_label
  • ifgt positive_label
  • ifle non_positive_label
Stack Effect:
  • Before: ..., value (int)
  • After: ... (the value is consumed)
