Bitwise XOR Instructions

{link:ixor} - 0x82 (130)
{link:lxor} - 0x83 (131)

Purpose

Performs a bitwise XOR operation on two values.
The result is pushed onto the operand stack.

  • {link:ixor}: performs a bitwise XOR on two int values
  • {link:lxor}: performs a bitwise XOR on two long values
Operands: None
Format: {instruction}
Examples:
  • ixor
  • lxor
Stack Effect:
  • ixor
    Before: ..., value1 (int), value2 (int)
    After: ..., result (int)
  • lxor
    Before: ..., value1 (long), value2 (long)
    After: ..., result (long)
