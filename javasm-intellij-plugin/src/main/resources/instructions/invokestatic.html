Invoke Static Method

{link:invokestatic} - 0xb8 (184)

Purpose

Invokes a static method without consuming an object reference. The call target is selected from the symbolic method reference after resolution.

Operands:
  • methodref (2 bytes): Index into the constant pool, pointing to a Methodref
Format:
invokestatic <methodref>
Methodref format:

The method reference format specifies the target method as follows:

<classname>-><methodname>(<argument_descriptor>)<return_descriptor>
  • <classname>: Fully qualified class name in JVM internal form (slash-separated), e.g., java/lang/Math
  • <methodname>: Method name, e.g., max, abs
  • <argument_descriptor>: JVM method argument descriptor, e.g., I (int), Ljava/lang/String; (String object)
  • <return_descriptor>: JVM return type descriptor, e.g., V (void), I (int), Ljava/lang/Object; (Object)
Examples:
  • invokestatic java/lang/Math->max(II)I
  • invokestatic my/pkg/Utils->doStaticThing()V
Stack Effect:

Before: ..., arg1, arg2, ..., argN
After: ..., returnValue? (if return type is non-void)

Pops all descriptor arguments. Pushes return value according to the declared return type (category-2 for long/double, category-1 otherwise).

Verification:
  • Verification checks argument types and stack shape against the method descriptor.
  • Before invocation, the target class or interface is initialised if required.
Notes:
  • No receiver object is consumed; all arguments are explicit in the descriptor.
  • Class initialisation side effects can execute user code before the target method body runs.
Runtime Exceptions and Errors:
  • LinkageError: if the symbolic method reference cannot be resolved or if the resolved method is not accessible.
  • IncompatibleClassChangeError: if the resolved method is not static or if the resolved method is not a valid method handle for the invocation type.
  • IllegalAccessError: if the resolved method is not accessible from the current class.
  • WrongMethodTypeException: if the actual argument types do not match the method descriptor at runtime.
