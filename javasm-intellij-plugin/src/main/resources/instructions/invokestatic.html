Invoke Static Method

{link:invokestatic} - 0xb8 (184)

Purpose

Invokes a static method of a class.
Since the method is static, no object reference is needed.

Operands:
  • methodref (2 bytes): Index into the constant pool, pointing to a Methodref
Format:
invokestatic <methodref>
Methodref format:

The method reference format specifies the target method as follows:

<classname>-><methodname>(<argument_descriptor>)<return_descriptor>
  • <classname>: Fully qualified class name in JVM internal form (slash-separated), e.g., java/lang/Math
  • <methodname>: Method name, e.g., max, abs
  • <argument_descriptor>: JVM method argument descriptor, e.g., I (int), Ljava/lang/String; (String object)
  • <return_descriptor>: JVM return type descriptor, e.g., V (void), I (int), Ljava/lang/Object; (Object)
Examples:
  • invokestatic java/lang/Math->max(II)I
  • invokestatic my/pkg/Utils->doStaticThing()V
Stack Effect:

Before: ..., arg1, arg2, ..., argN
After: ..., returnValue?

- Pops all arguments from the stack.
- Pushes return value if the method is non-void.
- Throws NoSuchMethodError if method not found.
- Throws IncompatibleClassChangeError if method is not static.
