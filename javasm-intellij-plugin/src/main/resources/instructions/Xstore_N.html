Purpose

Stores the top value from the operand stack into a local variable at index 0 to 3. The type of the value is indicated by the instruction prefix.

Operands: None
Format: {instruction}
Examples:
  • istore_1
  • astore_0
  • dstore_3
Stack Effect:
  • Before: ..., value
  • After: ... (value stored into local variable, value removed from stack)
