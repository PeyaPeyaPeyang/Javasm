Get Static Field Value Instruction

{link:getstatic} - 0xb2 (178)

Purpose

Fetches the value of a static field from a class and pushes it onto the operand stack.

Operands:
  • FieldDescriptor: Specifies the class, field name, and type descriptor of the static field to fetch. The format is Owner->FieldName:TypeDescriptor. The Owner does not include the L and ; typically used in type descriptors.
Format:

getstatic <FieldDescriptor>

Examples:
  • getstatic java/lang/System->out:Ljava/io/PrintStream;
  • getstatic java/lang/Math->PI:D
TypeDescriptor:

A TypeDescriptor specifies the type of the field. It must represent a valid Java type:

  • I: int
  • J: long
  • F: float
  • D: double
  • Z: boolean
  • B: byte
  • C: char
  • S: short
  • L<classname>>: A reference type (e.g., Ljava/lang/String;)
  • [: An array type (e.g., [I for int[])
Stack Effect:
  • Before: ...
  • After: ..., value (the value of the static field)
