Shift Right Instructions

{link:ishr} - 0x7a (122)
{link:lshr} - 0x7b (123)

Purpose

Shifts a value to the right logically by a specified number of bits.
The shift amount is masked: only the lowest 5 bits are used for ishr, and the lowest 6 bits for lshr.

  • ishr: logically right-shifts an int by an int amount
  • lshr: logically right-shifts a long by an int amount
Operands: None
Format: {instruction}
Examples:
  • ishr
  • lshr
Stack Effect:
  • ishr
    Before: ..., value (int), shift (int)
    After: ..., result (int)
  • lshr
    Before: ..., value (long), shift (int)
    After: ..., result (long)
