Double Compare Instructions

{link:dcmpg} - 0x98 (152)
{link:dcmpl} - 0x97 (151)

Purpose

Compares two double values from the operand stack. Pops the top two doubles and pushes an int result: -1 if value1 < value2, 0 if equal, 1 if value1 > value2.

The difference between dcmpg and dcmpl lies in handling NaN (Not-a-Number) values:

  • {link:dcmpg}: pushes 1 if either value is NaN.
  • {link:dcmpl}: pushes -1 if either value is NaN.
Operands: None
Format: {instruction}
Examples:
  • dcmpg
  • dcmpl
Stack Effect:
  • Before: ..., value1 (double), value2 (double)
  • After: ..., result (int) (-1, 0, or 1)
