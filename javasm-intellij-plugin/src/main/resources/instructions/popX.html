Stack Popping Instructions

{link:pop} - 0x57 (87)
{link:pop2} - 0x58 (88)

Purpose

Removes one or two values from the top of the operand stack.

Operands: None
Format: {instruction}
Stack Effect:
  • pop
    Before: ..., value
    After: ...
    (Removes one category-1 value: int, float, reference, etc.)
  • pop2
    Two valid use cases:
    - Case 1 (two category-1 values)
    Before: ..., value2, value1
    After: ...

    - Case 2 (one category-2 value)
    Before: ..., value (long or double)
    After: ...
Examples:
  • pop (e.g. discard an unused int)
  • pop2 (e.g. discard a double or two ints)
Notes:

Using pop on a category-2 value (like long or double) results in a VerifyError at runtime. Likewise, using pop2 when the stack does not match one of the expected patterns is invalid.
