Unconditional Branch Instructions

{link:goto} - 0xa7 (167)
{link:goto_w} - 0xc8 (200)

Purpose

Transfers control unconditionally to the instruction at the specified label.
goto supports labels within a 16-bit signed offset, while goto_w supports labels within a 32-bit signed offset.

Operands:
  • label: The target label to branch to.
Format:
  • goto <label>
  • goto_w <label>
Examples:
  • goto loop_start
  • goto_w end_of_method
