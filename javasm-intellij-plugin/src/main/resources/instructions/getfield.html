Get Instance Field Value Instruction

{link:getfield} - 0xb4 (180)

Purpose

Fetches the value of an instance field from an object reference and pushes it onto the operand stack.

Operands:
  • FieldDescriptor: Specifies the class, field name, and type descriptor of the instance field to fetch. The format is Owner->FieldName:TypeDescriptor. The Owner does not include the L and ; typically used in type descriptors.
Format:

getfield <FieldDescriptor>

Examples:
  • getfield java/lang/String->value:[C
  • getfield java/util/ArrayList->size:I
TypeDescriptor:

A TypeDescriptor specifies the type of the field. It must represent a valid Java type:

  • I: int
  • J: long
  • F: float
  • D: double
  • Z: boolean
  • B: byte
  • C: char
  • S: short
  • L<classname>: A reference type (e.g., Ljava/lang/String;)
  • [: An array type (e.g., [I for int[])
Stack Effect:
  • Before: ..., objectref
  • After: ..., value (the value of the instance field)

If objectref is null, a NullPointerException is thrown.
