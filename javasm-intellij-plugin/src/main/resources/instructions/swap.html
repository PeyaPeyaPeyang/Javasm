Stack Swap Instruction

{link:swap} - 0x5f (95)

Purpose

Swap the top two values on the operand stack. Both values must be category 1 values (i.e., occupy one stack word). Category 2 values such as long and double are not allowed and will cause a VerifyError at runtime.

Operands: None
Format: swap
Examples:
  • swap
Stack Effect:

Before: ..., value2, value1
After: ..., value1, value2

Notes:
  • Both value1 and value2 must be category 1: int, float, reference, or returnAddress.
  • Using swap with category 2 values (e.g. long, double) results in a VerifyError.
