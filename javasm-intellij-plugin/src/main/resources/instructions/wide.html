Widening Instruction

{link:wide} - 0xc4 (196)

Purpose

Modifies certain instructions to allow for a wider (16-bit) index or operand. This is used for accessing local variables or constants with indices larger than 255.

Operands:
  • Instruction: The instruction to be modified by wide.
  • Index: A 16-bit index or operand, depending on the instruction.
Format:

wide <Instruction> <Index>

Examples:
  • wide iload 300 (loads a local variable at index 300)
  • wide iinc 300 5 (increments the local variable at index 300 by 5)
Supported Instructions:

The following instructions can be modified by wide:

  • iload
  • fload
  • aload
  • lload
  • dload
  • istore
  • fstore
  • astore
  • lstore
  • dstore
  • ret
  • iinc
Stack Effect:

The stack effect depends on the instruction being modified by wide. For example:

  • wide iload:
    Before: ...
    After: ..., value
  • wide istore:
    Before: ..., value
    After: ...
