Bitwise OR Instructions

{link:ior} - 0x80 (128)
{link:lor} - 0x81 (129)

Purpose

Performs a bitwise OR operation on the top two values from the operand stack.
The result is pushed back onto the stack.

  • ior: operates on two int values
  • lor: operates on two long values
Operands: None
Format: {instruction}
Examples:
  • ior
  • lor
Stack Effect:
  • ior
    Before: ..., value1 (int), value2 (int)
    After: ..., result (int)
  • lor
    Before: ..., value1 (long), value2 (long)
    After: ..., result (long)
