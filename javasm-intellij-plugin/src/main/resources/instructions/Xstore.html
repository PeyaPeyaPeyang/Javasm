Store Instructions

{link:istore} - 0x36 (54)
{link:lstore} - 0x37 (55)
{link:fstore} - 0x38 (56)
{link:dstore} - 0x39 (57)
{link:astore} - 0x3a (58)

Purpose

Stores a value from the operand stack into a local variable at the specified index N.

Operands:
  • N: Local variable index (decimal or hexadecimal, e.g., 5 or 0x05)
Format: {instruction} N
Notes on wide prefix:

If the wide prefix is used, the index operand N is extended from 1 byte to 2 bytes, allowing for larger local variable indices.

Examples:
  • istore 3
  • lstore 0x0A
  • wide fstore 0x0100 (index 256)
Stack Effect:
  • istore, fstore, astore
    Before: ..., value
    After: ... (value stored in local variable N)
  • lstore, dstore
    Before: ..., value2, value1
    After: ... (value stored in local variable N)
    (Note: long and double use two stack words)
