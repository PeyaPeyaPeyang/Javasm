Create New Object Instruction

{link:new} - 0xbb (187)

Purpose

Allocates memory for a new object of the specified class. The object is not initialized; the constructor must be invoked explicitly using the invokespecial instruction.

Operands:
  • TypeDescriptor: Specifies the class of the object to be created. The format is L <classname>>; , where <classname> is the fully qualified name of the class (e.g., Ljava/lang/String;).
Format:

new <TypeDescriptor>

Examples:
  • new Ljava/lang/String; (creates a new String object)
  • new Ljava/util/ArrayList; (creates a new ArrayList object)
TypeDescriptor:

The TypeDescriptor specifies the class of the object to be created. It must represent a valid reference type:

  • L<classname>>;: A reference type (e.g., Ljava/lang/String;)
Stack Effect:

The operand stack is updated with a reference to the newly allocated object.

  • Before: ...
  • After: ..., objectref (a reference to the newly created object)

If the class cannot be found or instantiated, a ClassNotFoundException or InstantiationException is thrown.
