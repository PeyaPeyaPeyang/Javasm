Push Null Reference

{link:aconst_null} - 0x01 (1)

Purpose

Pushes a null reference onto the operand stack. This is typically used to initialize variables or as a placeholder for objects that are not yet assigned.

Operands: (none)
Format:

aconst_null

Examples:
  • aconst_null (pushes a null reference onto the stack)
Stack Effect:

The operand stack is updated with a null reference.

  • Before: ...
  • After: ..., null
Notes:

The aconst_null instruction is used for reference types only. It cannot be used for primitive types such as int or float. Attempting to use a null reference without proper handling may result in a NullPointerException.
