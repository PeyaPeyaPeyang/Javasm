Push Null Reference

{link:aconst_null} - 0x01 (1)

Purpose

Pushes the null reference onto the operand stack. In verifier terms, this value has the special null type, which is assignment-compatible with any reference type.

Operands: (none)
Format:

aconst_null

Examples:
  • aconst_null (pushes a null reference onto the stack)
Stack Effect:

The operand stack is updated with a null reference (category-1 value).

  • Before: ...
  • After: ..., null
Notes:

The aconst_null instruction is used for reference types only. It cannot be used for primitive types such as int or float.

Verification:
  • Verification requires that the operand stack can accept a reference value at the point of this instruction.
Runtime Exceptions and Errors:

This instruction does not throw any instruction-specific runtime exceptions.
