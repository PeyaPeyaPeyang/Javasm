Push Signed Short Instruction

{link:sipush} - 0x11 (17)

Purpose

Pushes a signed 16-bit short value (−32,768 to 32,767) onto the operand stack as an int. This instruction is used for values outside the range handled by {link:bipush} and {link:iconst_m1} to {link:iconst_5}.

Operands:
  • short: 16-bit signed immediate value to push onto the stack.
Format: sipush <short>
Examples:
  • sipush 1000
  • sipush -32000
Note:

For values between −128 and 127 inclusive, use bipush. For values between −1 and 5 inclusive, use iconst_m1 to iconst_5.

Stack Effect:
  • Before: ...
  • After: ..., value (int)
