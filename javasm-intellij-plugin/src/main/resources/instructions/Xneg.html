Negate Instructions

{link:ineg} - 0x74 (116)
{link:lneg} - 0x75 (117)
{link:fneg} - 0x76 (118)
{link:dneg} - 0x77 (119)

Purpose

Negates the top value on the operand stack (i.e., multiplies by -1) and pushes the result back on the stack.

Operands: None
Format: {instruction}
Examples:
  • ineg
  • dneg
Stack Effect:
  • Before: ..., value
  • After: ..., -value
