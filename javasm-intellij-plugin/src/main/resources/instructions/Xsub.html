Subtraction Instructions

{link:isub} - 0x64 (100)
{link:lsub} - 0x65 (101)
{link:fsub} - 0x66 (102)
{link:dsub} - 0x67 (103)

Purpose

Subtracts the top operand from the second-top operand on the stack and pushes the result, according to the instruction's type.

Operands: None
Format: {instruction}
Examples:
  • isub
  • dsub
Stack Effect:
  • Before: ..., value1, value2 (value1 - value2)
  • After: ..., result
