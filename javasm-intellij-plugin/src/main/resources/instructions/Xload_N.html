Purpose

Loads a value from a local variable index N (0 to 3) onto the operand stack. These instructions are shortcuts for {instruction} N without requiring an explicit operand.

Operands:

None. The index is implied by the instruction name.

Format: {instruction}
Examples:
  • iload_0
  • aload_3
  • dload_2
Stack Effect:
  • aload_<n>, iload_<n>, fload_<n>
    Before: ...
    After: ..., value (category-1 value)
  • lload_<n>, dload_<n>
    Before: ...
    After: ..., value (category-2 value, two stack slots)
Verification:
  • Verification requires that the operand stack can accept the appropriate category of value at the point of this instruction.
Notes:
  • These instructions are optimised shortcuts for loading from local variable indices 0 to 3. For indices 4 and above, the corresponding non-shortcut instructions (e.g. iload, aload, etc.) with explicit operands must be used.
Runtime Exceptions and Errors:

This instruction does not throw any instruction-specific runtime exceptions.
