Shift Left Instructions

{link:ishl} - 0x78 (120)
{link:lshl} - 0x79 (121)

Purpose

Shifts a value to the left logically by a specified number of bits.
The shift amount is masked: only the lowest 5 bits are used for ishl, and the lowest 6 bits for lshl.

  • ishl: logically left-shifts an int by an int amount
  • lshl: logically left-shifts a long by an int amount
Operands: None
Format: {instruction}
Examples:
  • ishl
  • lshl
Stack Effect:
  • ishl
    Before: ..., value (int), shift (int)
    After: ..., result (int)
  • lshl
    Before: ..., value (long), shift (int)
    After: ..., result (long)
