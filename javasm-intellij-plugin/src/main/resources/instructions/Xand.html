Bitwise AND Instructions

{link:iand} - 0x7e (126)
{link:land} - 0x7f (127)

Purpose

Performs a bitwise AND operation on the top two values from the operand stack. The result is pushed back onto the stack.

  • iand: operates on two int values
  • land: operates on two long values
Operands: None
Format: {instruction}
Examples:
  • iand
  • land
Stack Effect:
  • iand
    Before: ..., value1 (int), value2 (int) (category-1)
    After: ..., result (int) (category-1)
  • land
    Before: ..., value1 (long), value2 (long) (category-2)
    After: ..., result (long) (category-2)
Verification:
  • Verification requires that the operand stack can accept the appropriate category of values at the point of this instruction.
Notes:
  • These instructions perform a bitwise AND on the binary representations of the input values.
  • They do not perform any type conversion; the input values must already be of the correct type (int for iand and long for land).
Runtime Exceptions and Errors:

These instructions do not throw any instruction-specific runtime exceptions.
