Reference Comparison Instructions

{link:if_acmpeq} - 0xa5 (165)
{link:if_acmpne} - 0xa6 (166)

Purpose

Compares two object references from the operand stack and branches to the specified label based on the result.

  • if_acmpeq: Branches if the two references are equal.
  • if_acmpne: Branches if the two references are not equal.
Operands:
  • label: The target label to branch to if the condition is met.
Format:
  • if_acmpeq <label>
  • if_acmpne <label>
Examples:
  • if_acmpeq equal_label
  • if_acmpne not_equal_label
Stack Effect:
  • Before: ..., value1 (reference), value2 (reference)
  • After: ... (both values are consumed)
