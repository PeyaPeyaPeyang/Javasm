Array Store Instructions

{link:iastore} - 0x4f (79)
{link:lastore} - 0x50 (80)
{link:fastore} - 0x51 (81)
{link:dastore} - 0x52 (82)
{link:aastore} - 0x53 (83)
{link:bastore} - 0x54 (84)
{link:castore} - 0x55 (85)
{link:sastore} - 0x56 (86)

Purpose

Stores a value into an array. The array reference, index, and the value to store are popped from the operand stack.

  • {link:iastore}: stores int into int[]
  • {link:lastore}: stores long into long[]
  • {link:fastore}: stores float into float[]
  • {link:dastore}: stores double into double[]
  • {link:aastore}: stores reference into Object[]
  • {link:bastore}: stores byte or boolean into byte[]/boolean[]
  • {link:castore}: stores char into char[]
  • {link:sastore}: stores short into short[]

If the array reference is null or the index is out of bounds, an exception is thrown.

Operands: (none)
Format: {instruction}
Examples:
  • iastore
  • lastore
  • fastore
  • dastore
  • aastore
  • bastore
  • castore
  • sastore
Stack Effect:
  • iastore, fastore, aastore, bastore, castore, sastore
    Before: ..., arrayref, index, value
    After: ...
  • lastore, dastore
    Before: ..., arrayref, index, value2, value1
    After: ...
    (Note: long and double values occupy two stack words)
