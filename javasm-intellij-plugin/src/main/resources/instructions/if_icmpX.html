Integer Comparison Instructions

{link:if_icmpeq} - 0x9f (159)
{link:if_icmpne} - 0xa0 (160)
{link:if_icmplt} - 0xa1 (161)
{link:if_icmpge} - 0xa2 (162)
{link:if_icmpgt} - 0xa3 (163)
{link:if_icmple} - 0xa4 (164)

Purpose

Compares two int values from the operand stack using signed integer comparison and conditionally branches.

  • {link:if_icmpeq}: branches if value1 == value2.
  • {link:if_icmpne}: branches if value1 != value2.
  • {link:if_icmplt}: branches if value1 < value2.
  • {link:if_icmpge}: branches if value1 >= value2.
  • {link:if_icmpgt}: branches if value1 > value2.
  • {link:if_icmple}: branches if value1 <= value2.

Here, value2 is the top of stack, and value1 is the next value below it.

Operands:
  • label: target label encoded as a signed 16-bit offset relative to the current opcode.
Format:
  • if_icmpeq <label>
  • if_icmpne <label>
  • if_icmplt <label>
  • if_icmpge <label>
  • if_icmpgt <label>
  • if_icmple <label>
Examples:
  • if_icmpeq equal_label
  • if_icmpne not_equal_label
  • if_icmplt less_than_label
  • if_icmpge greater_or_equal_label
  • if_icmpgt greater_than_label
  • if_icmple less_or_equal_label
Stack Effect:
  • Before: ..., value1 (int), value2 (int)
  • After: ... (consumes two category-1 int values)
Notes:
  • Comparisons are signed; these opcodes do not provide unsigned comparison semantics.
  • If one of the values being compared is known to be zero, it may be more efficient to use the corresponding {link:ifX} instruction to compare a single int value against zero instead. Choose the appropriate instruction based on the context.
Verification:
  • Verification requires that the operand stack can accept two category-1 int values at the point of this instruction.
  • No constant-pool resolution is performed by these instructions.
  • Branch target and frame-type consistency are verified at class loading/linking time.
Runtime Exceptions and Errors:

These instructions do not throw any instruction-specific runtime exceptions. Invalid branch targets are rejected during verification.
