Integer Comparison Instructions

{link:if_icmpeq} - 0x9f (159)
{link:if_icmpne} - 0xa0 (160)
{link:if_icmplt} - 0xa1 (161)
{link:if_icmpge} - 0xa2 (162)
{link:if_icmpgt} - 0xa3 (163)
{link:if_icmple} - 0xa4 (164)

Purpose

Compare two integer values from the operand stack and branches to the specified label based on the result.

  • {link:if_icmpeq}: Branches if the two integers are equal.
  • {link:if_icmpne}: Branches if the two integers are not equal.
  • {link:if_icmplt}: Branches if the first integer is less than the second.
  • {link:if_icmpge}: Branches if the first integer is greater than or equal to the second.
  • {link:if_icmpgt}: Branches if the first integer is greater than the second.
  • {link:if_icmple}: Branches if the first integer is less than or equal to the second.
Operands:
  • label: The target label to branch to if the condition is met.
Format:
  • if_icmpeq <label>
  • if_icmpne <label>
  • if_icmplt <label>
  • if_icmpge <label>
  • if_icmpgt <label>
  • if_icmple <label>
Examples:
  • if_icmpeq equal_label
  • if_icmpne not_equal_label
  • if_icmplt less_than_label
  • if_icmpge greater_or_equal_label
  • if_icmpgt greater_than_label
  • if_icmple less_or_equal_label
Stack Effect:
  • Before: ..., value1 (int), value2 (int)
  • After: ... (both values are consumed)
