Jump to Subroutine Instructions

{link:jsr} - 0xa8 (168)
{link:jsr_w} - 0xc9 (201)

Purpose

Transfers control to a subroutine at the specified label and pushes the return address onto the operand stack.
jsr uses a 16-bit signed offset, while jsr_w uses a 32-bit signed offset.

Operands:
  • label: The target label to jump to.
Format:
  • jsr <label>
  • jsr_w <label>
Examples:
  • jsr subroutine_start
  • jsr_w long_subroutine_start
Stack Effect:
  • Before: ...
  • After: ..., returnAddress (reference)
