Addition Instructions

{link:iadd} - 0x60 (96)
{link:ladd} - 0x61 (97)
{link:fadd} - 0x62 (98)
{link:dadd} - 0x63 (99)

Purpose

Adds the top two operands on the operand stack and pushes the result, respecting the instruction's type.

Operands: None
Format: {instruction}
Examples:
  • iadd
  • dadd
Stack Effect:
  • Before: ..., value1, value2
  • After: ..., result
