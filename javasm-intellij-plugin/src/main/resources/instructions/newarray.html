Create New Array Instruction

{link:newarray} - 0xbc (188)

Purpose

Creates a new array of a specified type. The length of the array is popped from the operand stack, and a reference to the newly created array is pushed onto the stack.

Operands:
  • TypeDescriptor: Specifies the type of the array elements. The format is:
    • I: int
    • J: long
    • F: float
    • D: double
    • Z: boolean
    • B: byte
    • C: char
    • S: short
    • L<classname>;: A reference type (e.g., Ljava/lang/String;)
Format:

newarray <TypeDescriptor>

Examples:
  • newarray I (creates an array of int)
  • newarray Ljava/lang/String; (creates an array of String)
TypeDescriptor:

The TypeDescriptor specifies the type and structure of the array. It must represent a valid Java type:

  • [: Indicates an array type. Each [ adds a dimension to the array.
  • I: int
  • J: long
  • F: float
  • D: double
  • Z: boolean
  • B: byte
  • C: char
  • S: short
  • L<classname>;: A reference type (e.g., Ljava/lang/String;)
Stack Effect:

The operand stack must contain the size of the array to create. The size must be a non-negative integer.

  • Before: ..., count
  • After: ..., arrayref (a reference to the newly created array)

If count is negative, a NegativeArraySizeException is thrown.
