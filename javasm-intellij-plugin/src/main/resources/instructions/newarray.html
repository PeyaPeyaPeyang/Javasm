Create New Array Instruction

{link:newarray} - 0xbc (188)

Purpose

Creates a new one-dimensional array of a primitive component type. The length is popped from the operand stack, and an array reference is pushed.

Operands:
  • atype: unsigned byte selecting the primitive component type:
    • 4: boolean
    • 5: char
    • 6: float
    • 7: double
    • 8: byte
    • 9: short
    • 10: int
    • 11: long
Format:

newarray <atype>

Examples:
  • newarray 10 (creates int[])
  • newarray 7 (creates double[])
Type Scope:

newarray is for primitive arrays only. Use anewarray for reference-component arrays and multianewarray for multi-dimensional creation.

Stack Effect:

The operand stack must contain the size of the array to create. The size must be a non-negative integer.

  • Before: ..., count
  • After: ..., arrayref (category-1 reference)
Runtime Exceptions and Errors:
  • NegativeArraySizeException: if the length of the array is negative.
  • OutOfMemoryError: if the Java Virtual Machine cannot allocate the array because it is out of memory.
Verification:
  • Verification requires that the operand stack can accept an int value for the array size and that the atype operand is valid.
