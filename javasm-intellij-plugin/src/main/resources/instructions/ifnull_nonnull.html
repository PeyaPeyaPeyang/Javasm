Null Check Branch Instructions

{link:ifnull} - 0xc6 (198)
{link:ifnonnull} - 0xc7 (199)

Purpose

Tests whether the top value on the operand stack is null and branches to the specified label based on the result.

  • ifnull: Branches if the value is null.
  • ifnonnull: Branches if the value is not null.
Operands:
  • label: The target label to branch to if the condition is met.
Format:
  • ifnull <label>
  • ifnonnull <label>
Examples:
  • ifnull null_label
  • ifnonnull not_null_label
Stack Effect:
  • Before: ..., value (reference)
  • After: ... (the value is consumed)
