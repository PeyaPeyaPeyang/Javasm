Float Compare Instructions

{link:fcmpg} - 0x96 (150)
{link:fcmpl} - 0x95 (149)

Purpose

Compares two float values. Pops two values from the operand stack and pushes an int result:
-1 if value1 < value2, 0 if equal, 1 if value1 > value2.

If either value is NaN:
- {link:fcmpg} pushes 1
- {link:fcmpl} pushes -1

Operands: None
Format: {instruction}
Examples:
  • fcmpg
  • fcmpl
Stack Effect:
  • Before: ..., value1 (float), value2 (float)
  • After: ..., result (int) (-1, 0, or 1)
