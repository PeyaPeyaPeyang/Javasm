Float Compare Instructions

{link:fcmpg} - 0x96 (150)
{link:fcmpl} - 0x95 (149)

Purpose

Compares two float values. Pops two values from the operand stack and pushes an int result:
-1 if value1 < value2, 0 if equal, 1 if value1 > value2.

If either value is NaN:
- {link:fcmpg} pushes 1
- {link:fcmpl} pushes -1

Operands: None
Format: {instruction}
Examples:
  • fcmpg
  • fcmpl
Stack Effect:
  • Before: ..., value1 (category-1 float), value2 (category-1 float)
  • After: ..., result (category-1 int) (-1, 0, or 1)
Notes:
  • +0.0f and -0.0f compare as equal.
Verification:
  • Verification requires that the operand stack can accept two category-1 float values and one category-1 int result at the point of this instruction.
Runtime Exceptions and Errors:

These instructions do not throw any instruction-specific runtime exceptions.
