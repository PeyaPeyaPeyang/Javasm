Throw Exception

{link:athrow} - 0xbf (191)

Purpose

Throws the exception object currently on top of the operand stack. The object must be a reference to a throwable object (subclass of java.lang.Throwable).

Operands: None
Format: athrow
Examples:
  • athrow
Stack Effect:
  • Before: ..., throwable-instance
  • After: (no values) (method execution is terminated by throwing the exception)

Throws the exception referenced by the object on the stack. If the object is null, a NullPointerException is thrown instead.
