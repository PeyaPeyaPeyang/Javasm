Throw Exception

{link:athrow} - 0xbf (191)

Purpose

Throws the exception object currently on top of the operand stack. The object must be a reference to a throwable object (subclass of java.lang.Throwable).

Normal execution of the current instruction stream does not continue past athrow. The JVM begins exception handler lookup in the current frame and unwinds frames as needed.

Operands: None
Format: athrow
Examples:
  • athrow
Stack Effect:
  • Before: ..., throwable-instance
  • After: (no values) (method execution is terminated by throwing the exception)
Verification:
  • Verification requires that the operand stack can accept a reference value at the point of this instruction.
Runtime Exceptions and Errors:
  • NullPointerException: if the reference on top of the stack is null.
  • ClassCastException: if the reference on top of the stack is not an instance of java.lang.Throwable.
  • NoClassDefFoundError: if the class of the reference on top of the stack cannot be found.
  • IncompatibleClassChangeError: if the class of the reference on top of the stack is not a java.lang.Throwable.
