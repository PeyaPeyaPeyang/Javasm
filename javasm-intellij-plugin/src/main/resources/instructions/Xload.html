Load Instructions

{link:iload} - 0x15 (21)
{link:lload} - 0x16 (22)
{link:fload} - 0x17 (23)
{link:dload} - 0x18 (24)
{link:aload} - 0x19 (25)

Purpose

Loads a value from a local variable at index N onto the operand stack.

In bytecode encoding, N is an unsigned 8-bit index unless the instruction is prefixed with wide, in which case it is unsigned 16-bit.

Operands:
  • N: Local variable index (decimal or hexadecimal, e.g. 3 or 0x03)
Format: {instruction} N
Notes on wide prefix:

With the wide prefix, the operand N is extended from 1 byte to 2 bytes, allowing local variable indices larger than 255.

Examples:
  • iload 2
  • aload 0x0A
  • wide dload 0x0100 (index 256)
Stack Effect:
  • iload, fload, aload
    Before: ...
    After: ..., value (category-1)
  • lload, dload
    Before: ...
    After: ..., value (category-2)
    (Note: long and double occupy two stack slots)
Verification:
  • Verification requires the local slot type at N to match the selected load opcode.
Runtime Exceptions and Errors:
  • IndexOutOfBoundsException if the local variable index is out of range (e.g. negative or beyond the method's local variable array size).
