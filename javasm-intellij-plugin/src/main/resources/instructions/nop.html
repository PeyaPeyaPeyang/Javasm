NOP Instruction

{link:nop} - 0x00 (0)

Purpose

It does nothing. It is commonly used as a placeholder in transformed bytecode and for patch-friendly instruction layout.

Operands:

None

Format:

nop

Examples:
  • nop
Stack Effect:
  • Before: ...
  • After: ... (no stack or local changes)
Notes:
  • nop has no constant-pool interaction and no instruction-specific runtime exception.
  • Although semantically inert, it still has bytecode index impact and can affect branch offsets/debug mapping.
  • Do not use nop for switch padding; tableswitch/lookupswitch define their own padding rules.
Verification:
  • Verification does not require any specific operand stack or local variable state for nop.
Runtime Exceptions and Errors:

This instruction does not throw any instruction-specific runtime exceptions.
