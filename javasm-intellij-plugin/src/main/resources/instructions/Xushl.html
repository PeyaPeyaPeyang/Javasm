Unsigned Shift Left Instructions

{link:iushl} - 0x78 (120)
{link:lushl} - 0x79 (121)

Purpose

Shifts a value to the left logically (unsigned) by a specified number of bits.
The shift amount is masked: only the lowest 5 bits are used for {link:iushl}, and the lowest 6 bits for {link:lushl}.

  • {link:iushl}: logically unsigned left-shifts an int by an int amount
  • {link:lushl}: logically unsigned left-shifts a long by an int amount
Operands: None
Format: {instruction}
Examples:
  • iushl
  • lushl
Stack Effect:
  • iushl
    Before: ..., value (int), shift (int)
    After: ..., result (int)
  • lushl
    Before: ..., value (long), shift (int)
    After: ..., result (long)
