このインスペクションは，コード内のすべての命令を解析し，実行時に到達不能な命令を報告します。

ジャンプやメソッド終了などの無条件の制御移動によって制御フローが到達できない場合，その命令は到達不能と見なされます。

次の命令は，無条件に制御フローを終了させるものと見なされます。

  • goto
  • return
  • ireturn，lreturn，freturn，dreturn，areturn
  • athrow
  • tableswitch と lookupswitch（すべての分岐が別の場所へジャンプする場合）

注意: jsr（サブルーチンへのジャンプ）は，サブルーチンの完了後に実行が再開されるため，制御フローを終了させるものとは 見なされません。

このような終了命令の後に続き，かついずれのジャンプ先や例外ハンドラの対象でもない命令は，警告としてフラグ付けされます。
