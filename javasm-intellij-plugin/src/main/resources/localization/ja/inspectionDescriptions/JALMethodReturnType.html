このインスペクションは，ireturn，lreturn，freturn，dreturn，areturn のような型付きの命令を含む， return 命令を対象にします。

各 return 命令の型が，メソッド記述子で指定された戻り値型と一致しているかを確認します。 型が一致しない場合はエラーが報告されます。

たとえば，次のようなケースは無効です。
  • double を返すと宣言されたメソッドで ireturn を使用する。
  • int を返すメソッドで return を使用する。

メソッドの宣言上の戻り値型と，実際の return 命令の整合性を保つことは，有効なバイトコードのために不可欠です。
