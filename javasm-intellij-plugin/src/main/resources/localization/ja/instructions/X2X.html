目的

オペランド・スタック先頭の値を，あるプリミティブ型から別のプリミティブ型へ変換します。値はポップされ，変換後の値がプッシュされます。

変換規則:
  • 浮動小数点から整数への変換（f2i, f2l, d2i, d2l）は 0 方向への丸めです。
  • 浮動小数点から整数への変換では，NaN は 0 に変換されます。
  • 範囲外の浮動小数点値は，変換先型の最小値/最大値に丸め込まれます。
  • i2b，i2s，i2c は int からの縮小変換です。i2c は符号なし 16 ビット char へゼロ拡張します。
  • f2d は拡大変換です。d2f は精度を失う可能性があり，無限大になる場合があります。
オペランド: なし
形式: {instruction}
例:
  • d2i
  • i2l
  • f2d
スタック効果:
  • Before: ..., value (type1)
  • After: ..., value (type2)
カテゴリ/スロットに関する備考:
  • long/double を含む変換は，カテゴリ 2 値（スタックスロット 2 つ）を消費または生成します。
  • それ以外のプリミティブ変換結果はすべて カテゴリ 1 です。
検証:
  • この命令位置で，オペランド・スタックが変換元型の入力と変換先型の出力を受け入れ可能であることが必要です。
実行時例外とエラー:

これらの命令固有の実行時例外は発生しません。
