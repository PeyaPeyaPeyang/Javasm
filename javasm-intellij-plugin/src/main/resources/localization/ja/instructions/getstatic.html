静的フィールドの取得

{link:getstatic} - 0xb2 (178)

目的

クラスの静的フィールド値を取得し，オペランド・スタックへプッシュします。

フィールド読み取り前にシンボリックフィールド参照が解決され，必要に応じて宣言クラスまたはインターフェースが初期化されます。

オペランド:
  • FieldDescriptor: 取得対象静的フィールドのクラス，フィールド名，型記述子を指定します。 形式は Owner->FieldName:TypeDescriptor です。 Owner には型記述子で使う L と ; は含めません。
形式:

getstatic <FieldDescriptor>

例:
  • getstatic java/lang/System->out:Ljava/io/PrintStream;
  • getstatic java/lang/Math->PI:D
TypeDescriptor:

TypeDescriptor はフィールド型を示し，有効な Java 型である必要があります。

  • I: int
  • J: long
  • F: float
  • D: double
  • Z: boolean
  • B: byte
  • C: char
  • S: short
  • L<classname>;: 参照型（例: Ljava/lang/String;）
  • [: 配列型（例: [I は int[]）
スタック効果:
  • Before: ...
  • After: ..., value（静的フィールド値。多くの型は カテゴリ 1，long/double は カテゴリ 2）
検証:
  • この命令位置で，オペランド・スタックがフィールド型の値を受け入れ可能であることが必要です。
実行時例外とエラー:
  • IncompatibleClassChangeError: 解決先フィールドが static でない場合。
  • NoSuchFieldError，IllegalAccessError，NoClassDefFoundError などのリンケージエラー: シンボリックフィールド参照を解決できない場合，または解決先フィールドにアクセスできない場合。
