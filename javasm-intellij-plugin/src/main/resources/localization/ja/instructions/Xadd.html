加算

{link:iadd} - 0x60 (96)
{link:ladd} - 0x61 (97)
{link:fadd} - 0x62 (98)
{link:dadd} - 0x63 (99)

目的

オペランド・スタック先頭の 2 つの値を加算し，命令の型に応じた結果をスタックへプッシュします。

  • {link:iadd} と {link:ladd} は 2 の補数算術を用い，オーバーフロー時はラップアラウンドします。
  • {link:fadd} と {link:dadd} は IEEE 754 の規則（NaN 伝播，符号付きゼロの扱いを含む）に従います。
オペランド: なし
形式: {instruction}
例:
  • iadd
  • dadd
スタック効果:
  • {link:iadd}, {link:fadd}
    Before: ..., value1, value2（カテゴリ 1）
    After: ..., result（カテゴリ 1）
  • {link:ladd}, {link:dadd}
    Before: ..., value1, value2（カテゴリ 2 値）
    After: ..., result（カテゴリ 2）
検証:
  • この命令位置で，オペランド・スタックが命令に対応するカテゴリの値を受け入れ可能であることが必要です。
  • これらの命令は定数プール解決を行いません。
実行時例外とエラー:

これらの命令固有の実行時例外は発生しません。
