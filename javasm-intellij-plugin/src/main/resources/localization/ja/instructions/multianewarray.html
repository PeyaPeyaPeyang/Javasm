多次元配列の生成

{link:multianewarray} - 0xc5 (197)

目的

指定された型の多次元配列を新規作成します。初期化する次元のサイズはオペランド・スタックから取得されます。 生成されるのは指定された先頭次元までです。

オペランド:
  • indexbyte1, indexbyte2: 現在クラスの実行時定数プールにおけるインデックス。 配列型へのシンボリック参照を指します。
  • dimensions: 初期化する次元数を表す符号なし byte。
形式:

multianewarray <indexbyte1> <indexbyte2> <dimensions>

例:
  • multianewarray [[I 2
  • multianewarray [[[Ljava/lang/String; 2
TypeDescriptor:

参照先の定数プールエントリは，配列型記述子へのシンボリック参照である必要があります。 記述子は次の規則で表現されます。

  • [: 配列の次元を表す
  • プリミティブ型: I, J, F, D, Z, B, C, S
  • L<classname>;: 参照型
スタック効果:
  • Before: ..., count1, count2, ..., countN（すべて int, カテゴリ 1）
  • After: ..., arrayref（カテゴリ 1）

サイズ値は順に消費され，count1 が最外次元に対応します。

備考:
  • dimensions の値は，配列型記述子が持つ次元数以下でなければなりません。
  • 生成されるのは先頭 dimensions 次元のみです。残りの次元は未初期化で， それらの要素は null 参照になります。
  • JVM における多次元配列は「配列の配列」として実装されています。
実行時例外とエラー:
  • NegativeArraySizeException: いずれかの次元サイズが負の場合。
  • ClassNotFoundException（NoClassDefFoundError にラップ）: 配列型を解決できない場合。
  • IllegalAccessError: 参照先クラスまたはインターフェースへアクセスできない場合。
