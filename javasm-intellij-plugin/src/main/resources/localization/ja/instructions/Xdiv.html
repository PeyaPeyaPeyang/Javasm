除算

{link:idiv} - 0x6c (108)
{link:ldiv} - 0x6d (109)
{link:fdiv} - 0x6e (110)
{link:ddiv} - 0x6f (111)

目的

スタックの 2 番目の値を先頭の値で割り，結果をプッシュします。整数型では 0 除算で ArithmeticException が発生し，浮動小数点型では IEEE 754 に従って Infinity または NaN になります。

  • {link:idiv}/{link:ldiv}: 整数除算。結果は 0 方向へ切り捨てられます。
  • {link:fdiv}/{link:ddiv}: IEEE 754 算術（NaN 伝播，符号付き無限大/ゼロを含む）。
  • 整数の特殊ケース MIN_VALUE / -1 は MIN_VALUE（ラップアラウンド）になります。
オペランド: なし
形式: {instruction}
例:
  • idiv
  • ddiv
スタック効果:
  • Before: ..., value1, value2（value1 は被除数，value2 は除数）
  • After: ..., result（カテゴリは命令型に依存）
検証:
  • この命令位置で，オペランド・スタックが命令型に対応したカテゴリの値を受け入れ可能であることが必要です。
備考:
  • fdiv と ddiv は IEEE 754 の規則に従うため，0 除算は例外ではなく特定の値（Infinity または NaN）になります。
実行時例外:
  • ArithmeticException は idiv または ldiv で除数（先頭オペランド）が 0 の場合に発生します。
