ビット AND

{link:iand} - 0x7e (126)
{link:land} - 0x7f (127)

目的

オペランド・スタック先頭の 2 値に対してビット単位の AND 演算を行い， 結果をスタックへプッシュします。

  • iand: int 値 2 つを対象に演算
  • land: long 値 2 つを対象に演算
オペランド: なし
形式: {instruction}
例:
  • iand
  • land
スタック効果:
  • iand
    Before: ..., value1 (int), value2 (int)（カテゴリ 1）
    After: ..., result (int)（カテゴリ 1）
  • land
    Before: ..., value1 (long), value2 (long)（カテゴリ 2）
    After: ..., result (long)（カテゴリ 2）
検証:
  • この命令位置で，オペランド・スタックが命令に対応したカテゴリの値を受け入れ可能であることが必要です。
備考:
  • これらの命令は，入力値のビット表現に対して AND を適用します。
  • 型変換は行われないため，入力値はあらかじめ正しい型（iand は int，land は long）である必要があります。
実行時例外とエラー:

これらの命令固有の実行時例外は発生しません。
