配列からの読み出し

{link:iaload} - 0x2e (46)
{link:laload} - 0x2f (47)
{link:faload} - 0x30 (48)
{link:daload} - 0x31 (49)
{link:aaload} - 0x32 (50)
{link:baload} - 0x33 (51)
{link:caload} - 0x34 (52)
{link:saload} - 0x35 (53)

目的

配列から値を読み出してオペランド・スタックへロードします。配列参照とインデックスをスタックからポップし，指定した値をプッシュします。

  • {link:iaload}: int[] から int をロード
  • {link:laload}: long[] から long をロード
  • {link:faload}: float[] から float をロード
  • {link:daload}: double[] から double をロード
  • {link:aaload}: Object[] から参照をロード
  • {link:baload}: byte[]/boolean[] から byte または boolean をロード
  • {link:caload}: char[] から char をロード
  • {link:saload}: short[] から short をロード

baload は byte 値を int へ符号拡張し，boolean 配列からの読み出しでは 0 または 1 が得られます。caload は char を int へゼロ拡張し， saload は short を int へ符号拡張します。

配列参照が null の場合，またはインデックスが範囲外の場合は例外が発生します。

オペランド: なし
形式: {instruction}
例:
  • iaload
  • laload
  • faload
  • daload
  • aaload
  • baload
  • caload
  • saload
スタック効果:
  • iaload, faload, aaload, baload, caload, saload
    Before: ..., arrayref, index
    After: ..., value
  • laload, daload
    Before: ..., arrayref, index
    After: ..., value（カテゴリ 2）
    （備考: long と double はスタックスロットを 2 つ使用します）
検証:
  • この命令位置で，オペランド・スタックが配列参照とインデックス用の int 値をそれぞれ受け入れ可能であることが必要です。
実行時例外:
  • NullPointerException: arrayref が null の場合。
  • ArrayIndexOutOfBoundsException：index が範囲外の場合。
