目的

ローカル変数インデックス N（0〜3）から値をオペランド・スタックへロードします。 これらは明示的なオペランドを必要としない {/instruction} N の短縮命令です。

オペランド:

なし。インデックスは命令名に含まれます。

形式: {instruction}
例:
  • iload_0
  • aload_3
  • dload_2
スタック効果:
  • aload_<n>, iload_<n>, fload_<n>
    Before: ...
    After: ..., value（カテゴリ 1 値）
  • lload_<n>, dload_<n>
    Before: ...
    After: ..., value（カテゴリ 2 値。スタックスロット 2 つ）
検証:
  • この命令位置で，オペランド・スタックが適切なカテゴリの値を受け入れ可能であることが必要です。
備考:
  • これらはローカル変数インデックス 0〜3 の読み出しを最適化する短縮命令です。インデックスが 4 以上の場合は，iload や aload など，明示オペランド付きの通常命令を使用します。
実行時例外とエラー:

この命令固有の実行時例外は発生しません。
