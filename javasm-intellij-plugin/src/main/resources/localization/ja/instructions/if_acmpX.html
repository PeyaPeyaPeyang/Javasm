参照の比較

{link:if_acmpeq} - 0xa5 (165)
{link:if_acmpne} - 0xa6 (166)

目的

2 つの参照値を 参照同一性（同一オブジェクトか，或いは両方 null か）で比較し， 条件に応じて分岐します。

  • {link:if_acmpeq}: 2 つの参照が同一であれば分岐。
  • {link:if_acmpne}: 2 つの参照が同一でなければ分岐。

これは内容比較ではありません。equals() は呼び出しません。

オペランド:
  • label: 現在命令位置を基準とする符号付き 16 ビット相対オフセットで表される分岐先ラベル。
形式:
  • if_acmpeq <label>
  • if_acmpne <label>
例:
  • if_acmpeq same_ref（同一参照なら真。両方 null も含む）
  • if_acmpne different_ref（参照同一性が異なるとき真）
スタック効果:
  • Before: ..., value1 (reference), value2 (reference)
  • After: ...（カテゴリ 1 の参照値を 2 つ消費）
検証:
  • この命令位置で，オペランド・スタックが カテゴリ 1 の参照値 2 つを受け入れ可能であることが必要です。
  • 分岐先妥当性とフレーム型整合性は，クラスのロード/リンク時に検証されます。
実行時例外とエラー:

これらの命令固有の実行時例外は発生しません。不正な分岐先は検証時に拒否されます。
