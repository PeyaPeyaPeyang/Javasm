スタック値の入れ替え

{link:swap} - 0x5f (95)

目的

オペランド・スタック先頭の 2 値を入れ替えます。両方とも category 1 値（スタックワード 1 個）である必要があります。 long や double のような category 2 値は使用できず，不正な場合は実行時に VerifyError となります。

オペランド: なし
形式: swap
例:
  • swap
スタック効果:

Before: ..., value2, value1
After: ..., value1, value2

備考:
  • value1 と value2 は，いずれも category 1（int, float, 参照，returnAddress）である必要があります。
  • category 2 値（例: long, double）に swap を適用すると検証失敗となり，通常はロード/リンク時に VerifyError として現れます。
検証:
  • この命令位置で，オペランド・スタックが カテゴリ 1 値 2 つを受け入れ可能であることが必要です。
  • カテゴリ 2 値は許可されず，検証は失敗します。
実行時例外とエラー:

この命令固有の実行時例外は発生しません。
