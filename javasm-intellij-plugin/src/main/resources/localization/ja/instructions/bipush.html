byte 値のプッシュ

{link:bipush} - 0x10 (16)

目的

符号付き byte 値（-128 〜 127）を int としてオペランド・スタックにプッシュします。この命令は iconst_m1 ～ iconst_5 で扱えない値をロードするときに使用します。

即値の 1 byte オペランドは，プッシュ前に符号拡張されて 32 ビットの int になります。

オペランド:
  • byte: スタックへプッシュする 8 ビット符号付き即値。
形式: bipush <byte>
例:
  • bipush 100
  • bipush -128
スタック効果:
  • Before: ...
  • After: ..., value（カテゴリ 1 の int）
検証:
  • この命令位置で，オペランド・スタックが int 値を受け入れ可能であることが必要です。
備考:

-1 から 5 までの値には，bipush ではなく対応する iconst_m1 から iconst_5を使用してください。

実行時例外とエラー:

この命令固有の実行時例外は発生しません。
