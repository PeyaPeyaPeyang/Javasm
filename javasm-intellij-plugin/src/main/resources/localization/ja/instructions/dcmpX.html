double の比較

{link:dcmpg} - 0x98 (152)
{link:dcmpl} - 0x97 (151)

目的

オペランド・スタック上の 2 つの double 値を比較します。先頭 2 値をポップし，結果として int をプッシュします。結果は value1 < value2 なら -1，等しければ 0，value1 > value2 なら 1 です。

dcmpg と dcmpl の違いは，NaN（Not-a-Number）の扱いです。

  • {link:dcmpg}: いずれかの値が NaN の場合，1 をプッシュします。
  • {link:dcmpl}: いずれかの値が NaN の場合，-1 をプッシュします。
オペランド: なし
形式: {instruction}
例:
  • dcmpg
  • dcmpl
スタック効果:
  • Before: ..., value1 (カテゴリ 2 double), value2 (カテゴリ 2 double)
  • After: ..., result (カテゴリ 1 int)（-1, 0, 1）
備考:
  • +0.0 と -0.0 は等しいものとして比較されます。
検証:
  • この命令位置で，オペランド・スタックが カテゴリ 2 の double 値 2 つを受け入れ，かつ カテゴリ 1 の int 結果を返せることが必要です。
実行時例外とエラー:

この命令固有の実行時例外は発生しません。
