スタックの複製

{link:dup} - 0x59 (89)
{link:dup_x1} - 0x5a (90)
{link:dup_x2} - 0x5b (91)
{link:dup2} - 0x5c (92)
{link:dup2_x1} - 0x5d (93)
{link:dup2_x2} - 0x5e (94)

目的

オペランド・スタック上の 1 つまたは 2 つの値を複製し，必要に応じてより深い位置へ挿入します。

オペランド: なし
形式: {instruction}
スタック効果:
  • {link:dup}
    Before: ..., value1（value1 は カテゴリ 1）
    After: ..., value1, value1
  • {link:dup_x1}
    Before: ..., value2, value1（両方 カテゴリ 1）
    After: ..., value1, value2, value1
  • {link:dup_x2}
    Form 1（すべて カテゴリ 1）:
    Before: ..., value3, value2, value1
    After: ..., value1, value3, value2, value1
    Form 2:
    Before: ..., value2, value1（value2 は カテゴリ 2，value1 は カテゴリ 1）
    After: ..., value1, value2, value1
  • {link:dup2}
    Form 1:
    Before: ..., value2, value1（両方 カテゴリ 1）
    After: ..., value2, value1, value2, value1
    Form 2:
    Before: ..., value1（カテゴリ 2）
    After: ..., value1, value1
  • {link:dup2_x1}
    Form 1（すべて カテゴリ 1）:
    Before: ..., value3, value2, value1
    After: ..., value2, value1, value3, value2, value1
    Form 2:
    Before: ..., value2, value1（value2 は カテゴリ 1，value1 は カテゴリ 2）
    After: ..., value1, value2, value1
  • {link:dup2_x2}
    Form 1（すべて カテゴリ 1）:
    Before: ..., value4, value3, value2, value1
    After: ..., value2, value1, value4, value3, value2, value1
    Form 2:
    Before: ..., value3, value2, value1（value3/value2 は カテゴリ 1，value1 は カテゴリ 2）
    After: ..., value1, value3, value2, value1
    Form 3:
    Before: ..., value3, value2, value1（value3 は カテゴリ 2，value2/value1 は カテゴリ 1）
    After: ..., value2, value1, value3, value2, value1
    Form 4:
    Before: ..., value2, value1（両方 カテゴリ 2）
    After: ..., value1, value2, value1
例:
  • dup
  • dup_x1
  • dup2（long 1 つ，または int 2 つを対象）
検証:
  • この命令位置で，オペランド・スタックが必要な形状・カテゴリを満たし，複製後のスタック形も妥当であることが必要です。
実行時例外とエラー:
  • これらの命令固有の実行時例外は発生しません。
