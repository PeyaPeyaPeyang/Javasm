{link:dconst_1} - 0x0f (15)

目的

指定されたプリミティブ型の定数値をオペランド・スタックへプッシュします。

  • iconst_m1 〜 iconst_5: int 定数 -1 〜 5 をプッシュします。
  • lconst_0, lconst_1: long の 0 と 1 をプッシュします。
  • fconst_0, fconst_1, fconst_2: float の 0.0, 1.0, 2.0 をプッシュします。
  • dconst_0, dconst_1: double の 0.0 と 1.0 をプッシュします。
オペランド: なし
形式: {instruction}
例:
  • iconst_3（int 3 をプッシュ）
  • lconst_1（long 1 をプッシュ）
  • dconst_1（double 1.0 をプッシュ）
  • fconst_0（float 0.0 をプッシュ）
スタック効果:
  • Before: ...
  • After: ..., value（int/float は カテゴリ 1，long/double は カテゴリ 2）
検証:
  • この命令位置で，各命令が生成する値カテゴリをオペランド・スタックが受け入れ可能であることが必要です。
備考:
  • これらの命令は，頻繁に使用される定数向けに最適化されており，該当値の定数プール参照を不要にします。
  • byte 範囲の定数には bipush，short 範囲の定数には sipush を使用します。その他の値は ldc などの関連命令で定数プールからロードします。
