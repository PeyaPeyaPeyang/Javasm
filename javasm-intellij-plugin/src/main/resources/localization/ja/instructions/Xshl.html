左シフト

{link:ishl} - 0x78 (120)
{link:lshl} - 0x79 (121)

目的

指定ビット数だけ値を論理左シフトします。
シフト量はマスクされ，ishl では下位 5 ビット，lshl では下位 6 ビットのみが使用されます。

  • ishl: int を int 量だけ論理左シフト
  • lshl: long を int 量だけ論理左シフト
オペランド: なし
形式: {instruction}
例:
  • ishl
  • lshl
スタック効果:
  • ishl
    Before: ..., value (int), shift (int)（両方 カテゴリ 1）
    After: ..., result (int)（カテゴリ 1）
  • lshl
    Before: ..., value (long), shift (int)（value は カテゴリ 2）
    After: ..., result (long)（カテゴリ 2）
検証:
  • この命令位置で，オペランド・スタックが命令に対応したカテゴリの値を受け入れ可能であることが必要です。
備考:
  • これらは論理左シフトであり，右側には 0 ビットが挿入されます。符号ビットは特別扱いされません。
  • シフト量は値のビット幅を超えないようにマスクされます。ishl は実質 32 での剰余，lshl は実質 64 での剰余として扱われます。
実行時例外とエラー:

これらの命令固有の実行時例外は発生しません。
