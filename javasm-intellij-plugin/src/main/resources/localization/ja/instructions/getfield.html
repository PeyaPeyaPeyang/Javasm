インスタンス・フィールドの取得

{link:getfield} - 0xb4 (180)

目的

オブジェクト参照からインスタンス・フィールド値を取得し，オペランド・スタックへプッシュします。

対象フィールドは，実行時定数プールのシンボリックフィールド参照を解決して決定されます。

オペランド:
  • FieldDescriptor: 取得対象インスタンス・フィールドのクラス，フィールド名，型記述子を指定します。 形式は Owner->FieldName:TypeDescriptor です。 Owner には型記述子で使う L と ; は含めません。
形式:

getfield <FieldDescriptor>

例:
  • getfield java/lang/String->value:[C
  • getfield java/util/ArrayList->size:I
TypeDescriptor:

TypeDescriptor はフィールド型を示し，有効な Java 型である必要があります。

  • I: int
  • J: long
  • F: float
  • D: double
  • Z: boolean
  • B: byte
  • C: char
  • S: short
  • L<classname>;: 参照型（例: Ljava/lang/String;）
  • [: 配列型（例: [I は int[]）
スタック効果:
  • Before: ..., objectref
  • After: ..., value（フィールド値。多くの型は カテゴリ 1，long/double は カテゴリ 2）
検証:
  • 検証では，参照互換の入力が必要であり，後続フレームに対してフィールド型の値が保証されます。
実行時例外とエラー:
  • NullPointerException: オブジェクト参照が null の場合。
  • IncompatibleClassChangeError: 解決先フィールドが static の場合。
  • NoSuchFieldError，IllegalAccessError，NoClassDefFoundError などのリンケージエラー: シンボリックフィールド参照を解決できない場合，または解決先フィールドにアクセスできない場合。
