float の比較

{link:fcmpg} - 0x96 (150)
{link:fcmpl} - 0x95 (149)

目的

2 つの float 値を比較します。オペランド・スタックから 2 値をポップし， 結果として int をプッシュします。
value1 < value2 なら -1，等しければ 0，value1 > value2 なら 1 です。

いずれかの値が NaN の場合:
- {link:fcmpg} は 1 を push
- {link:fcmpl} は -1 を push

オペランド: なし
形式: {instruction}
例:
  • fcmpg
  • fcmpl
スタック効果:
  • Before: ..., value1 (カテゴリ 1 float), value2 (カテゴリ 1 float)
  • After: ..., result (カテゴリ 1 int)（-1, 0, 1）
備考:
  • +0.0f と -0.0f は等しいものとして比較されます。
検証:
  • この命令位置で，オペランド・スタックが カテゴリ 1 の float 値 2 つを受け入れ，かつ カテゴリ 1 の int 結果を返せることが必要です。
実行時例外とエラー:

これらの命令固有の実行時例外は発生しません。
