スタックからのポップ

{link:pop} - 0x57 (87)
{link:pop2} - 0x58 (88)

目的

オペランド・スタック先頭から 1 つまたは 2 つの値を取り除きます。

オペランド: なし
形式: {instruction}
スタック効果:
  • pop
    Before: ..., value
    After: ...
    （カテゴリ 1 値を 1 つ取り除く: int, float, 参照型など）
  • pop2
    有効な使用形は次の 2 つです。
    - Case 1（カテゴリ 1 値を 2 つ）
    Before: ..., value2, value1
    After: ...

    - Case 2（カテゴリ 2 値を 1 つ）
    Before: ..., value（long または double）
    After: ...
例:
  • pop（例: 不要な int 値を破棄）
  • pop2（例: double 1 つ，または int 2 つを破棄）
備考:
  • pop はスタック先頭が カテゴリ 1 の場合のみ有効です。
  • pop2 は上記 2 形のみ有効です。
  • カテゴリやスタック形が不正な場合，バイトコード検証で拒否されます（通常はロード/リンク時に VerifyError として現れます）。
  • これらの命令は定数プール解決を行わず，命令固有の実行時例外はありません。
検証:
  • この命令位置で，オペランド・スタックが命令に対応したカテゴリの値を受け入れ可能であることが必要です。
実行時例外とエラー:

これらの命令固有の実行時例外は発生しません。
