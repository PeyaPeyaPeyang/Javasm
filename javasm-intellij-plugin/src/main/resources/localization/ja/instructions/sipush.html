符号付き short 値のプッシュ

{link:sipush} - 0x11 (17)

目的

符号付き 16 ビット short 値（-32,768 〜 32,767）を int としてオペランド・スタックにプッシュします。この 命令は {link:bipush} および {link:iconst_m1} から {link:iconst_5} では扱えない範囲の値に使用します。

即値の 2 byte オペランドは，プッシュ前に符号拡張されて 32 ビット整数になります。

オペランド:
  • short: スタックへプッシュする 16 ビット符号付き即値。
形式: sipush <short>
例:
  • sipush 1000
  • sipush -32000
備考:

-128 〜 127 の値は bipush を使用します。-1 〜 5 の値は iconst_m1 から iconst_5 を使用します。

スタック効果:
  • Before: ...
  • After: ..., value（カテゴリ 1 の int）
検証:
  • この命令位置で，オペランド・スタックが カテゴリ 1 の int 値 1 つを受け入れ可能であることが必要です。
実行時例外とエラー:

この命令固有の実行時例外は発生しません。
