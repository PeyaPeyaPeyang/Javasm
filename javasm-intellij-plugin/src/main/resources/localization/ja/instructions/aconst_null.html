null 参照のプッシュ

{link:aconst_null} - 0x01 (1)

目的

オペランド・スタックに null 参照をプッシュします。検証器の観点では，この値は特別な null 型 として扱われ，任意の参照型に代入可能です。

オペランド: なし
形式:

aconst_null

例:
  • aconst_null（null 参照をスタックへプッシュ）
スタック効果:

オペランド・スタックに null 参照（カテゴリ 1 値）が追加されます。

  • Before: ...
  • After: ..., null
備考:

aconst_null 命令は参照型にのみ使用できます。int や float といった プリミティブ型には使用できません。

検証:
  • この命令位置で，オペランド・スタックが参照値を受け入れ可能であることが必要です。
実行時例外とエラー:

この命令固有の実行時例外は発生しません。
