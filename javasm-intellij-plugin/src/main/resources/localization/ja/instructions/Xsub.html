減算

{link:isub} - 0x64 (100)
{link:lsub} - 0x65 (101)
{link:fsub} - 0x66 (102)
{link:dsub} - 0x67 (103)

目的

オペランド・スタックの 2 番目の値から先頭の値を減算し，結果をスタックへプッシュします。 実際に行われる演算は命令種別によって異なります。

  • isub: int 値同士の減算
  • lsub: long 値同士の減算
  • fsub: float 値同士の減算
  • dsub: double 値同士の減算
オペランド: なし
形式: {instruction}
例:
  • isub
  • dsub
スタック効果:
  • isub
    Before: ..., value1 (int), value2 (int)（カテゴリ 1）
    After: ..., result (int)（カテゴリ 1）
    Operation: value1 - value2
  • lsub
    Before: ..., value1 (long), value2 (long)（カテゴリ 2）
    After: ..., result (long)（カテゴリ 2）
    Operation: value1 - value2
  • fsub
    Before: ..., value1 (float), value2 (float)（カテゴリ 1）
    After: ..., result (float)（カテゴリ 1）
    Operation: value1 - value2
  • dsub
    Before: ..., value1 (double), value2 (double)（カテゴリ 2）
    After: ..., result (double)（カテゴリ 2）
    Operation: value1 - value2
備考:
  • 整数減算（isub, lsub）は 2 の補数演算です。オーバーフローは無視され，結果は循環します。
  • 浮動小数点減算（fsub, dsub）は IEEE 754 に従います。
    • いずれかのオペランドが NaN の場合，結果は NaN になります。
    • 無限大を含む減算も IEEE 754 規則で処理されます。
    • 必要に応じて符号付きゼロが保持されます（例: +0.0 - +0.0 = +0.0）。
検証:
  • この命令位置で，命令型に対応したカテゴリの値をオペランド・スタックが受け入れ可能であることが必要です。
実行時例外とエラー:

これらの命令固有の実行時例外は発生しません。
