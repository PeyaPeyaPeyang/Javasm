This inspection targets local variable load and store instructions, such as iload, lload, fload, dload, aload, and their corresponding xstore forms.

When the instruction uses a variable index between 0 and 3, a short-form variant is available (e.g., iload_0 instead of iload 0).

This inspection detects cases where the long form is used unnecessarily and issues a warning. A Quick Fix is provided to replace the instruction with the equivalent short form.

The following replacements are suggested:

  • fstore 2 -< fstore_2, and so on.

Using the short form reduces bytecode size and improves readability.
