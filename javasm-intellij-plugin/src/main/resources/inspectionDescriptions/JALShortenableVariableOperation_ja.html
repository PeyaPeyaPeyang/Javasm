このインスペクションは，iload，lload，fload，dload，aload といったローカル変数の読み込み／書き込み命令，および対応する xstore 形式を対象にします。

命令が 0 から 3 の変数インデックスを使用している場合，短縮形のバリアントが利用できます（たとえば，iload 0 の代わりに iload_0）。

このインスペクションは，長い形式が不要に使われているケースを検出し，警告を出します。等価な短縮形に置き換えるクイックフィックスも提供されます。

次の置き換えが推奨されます。
  • iload 0 -< iload_0，iload 1 → iload_1 など。
  • fstore 2 -< fstore_2，以下同様です。

短縮形を使用すると，バイトコードサイズが小さくなり，可読性も向上します。
