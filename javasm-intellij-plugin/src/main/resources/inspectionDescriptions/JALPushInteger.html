This inspection audits instructions that push integer values onto the stack.

The targeted instructions are:
  • bipush X: If the value X fits within the range of iconst_X, it is recommended to use the corresponding iconst instruction instead.
  • sipush X: It should be replaced by bipush or iconst instructions when applicable.
The iconst instructions include:
  • iconst_m1 for -1
  • iconst_0 for 0
  • iconst_1 for 1
  • iconst_2 for 2
  • iconst_3 for 3
  • iconst_4 for 4
  • iconst_5 for 5

When a less efficient instruction is detected, a weak warning is reported. A Quick Fix is provided to replace it with the appropriate iconst or bipush instruction.
