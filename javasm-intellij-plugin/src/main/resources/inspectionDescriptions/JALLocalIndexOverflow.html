This inspection targets local variable operations, including Xload, Xstore, and iinc instructions.

It checks whether the referenced local variable index exceeds the one-byte range (0â€“255). If it does, the instruction is considered invalid unless it is prefixed with the wide modifier.

The following instruction types are subject to this inspection:

  • iload, lload, fload, dload, aload
  • istore, lstore, fstore, dstore, astore
  • iinc
  • ret

When a variable index exceeds 255, an error is reported and a Quick Fix is available to automatically add the wide prefix to the instruction.
